## Managing Customers

### Adding a customer

Navigate to **Sales → Customers → Add Customer**. Customers are tagged as **Individual** or **Business**, and each gets a sequential identifier (`CUS-001`).

### Detail page

Standard detail layout — icon-only breadcrumb, profile header, quick stats and underline tabs for **Overview**, **Orders**, **Invoices** and **Activity**. Internal notes can be added per customer.

### Use in orders

When creating a manual order or invoice, search by name, email or CUS-ID, or create a customer inline through the EntityPicker.
