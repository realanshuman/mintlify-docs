## Public Store Page

### Editing the slug

Go to **Settings → Store Page**. Edit the `token_id` slug — your storefront URL becomes `tokensupply.in/s/{token_id}` (or your custom domain once verified).

### Auto-save

Most fields auto-save on blur. The store page can be toggled public or private using the visibility switch — when private the page is hidden but tokenised customer portal links continue to work.
