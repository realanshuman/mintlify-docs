## Settings Overview

The Settings module (sidebar → **Settings**) has its own isolated layout with five horizontal tabs:

| Tab | What lives here |
| --- | --- |
| **General** | Business details, branding (logo + colours), Order ID Format, store address and contact |
| **Email Setup** | SMTP host, credentials, from-name / from-email, test send |
| **Domain** | Custom domains and DNS verification |
| **Notifications** | Per-event email preferences and digest frequency |
| **Store Page** | Public storefront slug and visibility |

\> Note: **Templates** and **Billing & Plans** are separate top-level items in the sidebar — not Settings tabs.
