## Dashboard Tour

The dashboard is your daily command centre. It surfaces the metrics that matter most and a quick-access toolbar.

### Primary KPI cards

Five cards summarise the period in store currency:

- **Revenue** — net of refunds, completed orders only
- **Orders** — completed transactions across all channels
- **Active Inventory** — keys currently available for sale
- **Open Tickets** — issues awaiting action
- **Credits Remaining** — billing usage at a glance

All numeric values use tabular figures so columns line up cleanly.

### Quick actions

Create a product, add inventory, log an order or open a ticket directly from the dashboard.

### Sidebar navigation

The flattened sidebar is organised into three sections:

- **Main** — Dashboard, Products, Purchases (Purchase Orders, Vendors), Sales (Customers, Invoices), Tickets, Messages, Orders (All Orders, Create Order), Analytics, Reports
- **Sales Channels** — Integrations, Sales Channel
- **Store** — Team, Settings, Templates, Billing & Plans

Hover the sidebar edge to minimise it. **Universal Search** (⌘K / Ctrl+K) opens a 360px command palette so you can jump to any product, vendor, customer, PO or transaction in one keystroke.
