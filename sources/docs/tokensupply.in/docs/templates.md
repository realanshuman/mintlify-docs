## Templates

Manage reusable email and document templates from the top-level **Templates** item in the sidebar.

### Merge fields

Use placeholders like `{{delivery_link}}`, `{{customer_name}}`, `{{order_id}}`, `{{key}}` — they are replaced at send time.

### Attachments

Templates handle MIME types correctly so PDFs, images and text files render properly in inboxes.

### Editor

The template editor supports rich text, plain HTML and live previews so you can confirm how an email will look before saving.
